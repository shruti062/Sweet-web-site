# 🍬 The Mithai Zone - Sweet Shop Website

A **responsive and modern sweet shop website** showcasing traditional Indian sweets, gallery, videos, and contact information. Built with **HTML, CSS, Bootstrap 4, and JavaScript**, this project highlights clean design, interactive carousel, and mobile-friendly layout.

---

## 🚀 Features

- **Home Section:** Hero carousel highlighting sweet images and promotions.  
- **Sweets Section:** Display of popular sweets like Barfi, Gulab Jamun, Rasgulla, Soan Papdi.  
- **About Section:** Story of the sweet shop, its tradition, and specialty.  
- **Gallery:** Showcase of sweets and festive preparations.  
- **Videos Section:** Embedded YouTube videos showcasing sweets preparation and reviews.  
- **Contact Section:** Form for customer inquiries, office address, email, phone, and timings.  
- **Footer:** Quick links, contact details, and social media icons.  
- **Responsive Design:** Fully mobile-friendly and optimized for desktops and tablets.  

---

## 🛠️ Technologies Used

- **HTML5** – structure of the website  
- **CSS3 & Bootstrap 4** – styling, layout, responsiveness, and animations  
- **JavaScript** – interactive carousel and dynamic elements  
- **Font Awesome** – icons for social media and search
